# End-to-End Pipeline Specification

## 1. Pipeline overview

The implementation must follow:

```text
Input ROI image + optional prior labels
    ↓
Preprocessing
    ├── grayscale 224×224 image for Gabor
    └── RGB 224×224×3 tensor for Conformer
    ↓
Gabor feature extraction
    ↓
Conformer feature extraction
    ↓
KCCA feature fusion
    ↓
Knowledge graph candidate filtering
    ↓
Cosine similarity matching
    ↓
Predicted palm ID
```

## 2. Training-time pipeline

```text
raw/ROI images
→ metadata normalization
→ per-palm 1:1 train/test split
→ train Conformer classifier on train split
→ extract Gabor features for train/test
→ extract Conformer features for train/test
→ fit KCCA on train features only
→ transform train/test features
→ build knowledge graph from train fused features
→ evaluate on test fused features
```

## 3. Inference-time pipeline

```python
def predict(image, gender=None, hand_side=None):
    gray = preprocess_for_gabor(image)
    rgb = preprocess_for_conformer(image)

    x_gabor = gabor.extract(gray)
    y_conformer = conformer.extract_features(rgb)

    z = kcca.transform_single(x_gabor, y_conformer)

    candidates = graph.query(gender=gender, hand_side=hand_side)
    if not candidates:
        candidates = graph.all_templates()

    prediction = cosine_matcher.match(z, candidates)
    return prediction
```

## 4. Required train/test isolation

The agent must prevent test leakage:

- Fit image normalizers/statistical scalers on train data only when learned.
- Train Conformer only on train split.
- Fit KCCA only on train features.
- Build graph only from train fused features.
- Use test split only for final evaluation.

## 6. Feature alignment contract

All feature `.npz` files must contain:

```python
features: np.ndarray
sample_ids: np.ndarray
palm_ids: np.ndarray
subject_ids: np.ndarray
gender: np.ndarray
hand_side: np.ndarray
image_paths: np.ndarray
split: str
dataset: str
config_hash: str
```

Before KCCA:

```python
assert list(gabor.sample_ids) == list(conformer.sample_ids)
```

Before graph build:

```python
assert len(fused_train.features) == len(fused_train.metadata)
```

## 7. Modes

Required modes:

1. `gabor_only`
2. `conformer_only`
3. `fused_kcca`
4. `one_stage_global`
5. `two_stage_graph`

Default production mode:

```text
fused_kcca + two_stage_graph + cosine matcher
```
