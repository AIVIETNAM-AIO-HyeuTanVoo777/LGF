# Results Directory

This directory is reserved for regenerated results from the restart experiments.

Old result summaries from previous protocol iterations were removed to avoid mixing obsolete evidence with the new protocol-sensitive evaluation.

New results should be generated by:

```bat
python scripts\aggregate_results.py
```
